# EcommerceProject

A production-grade eCommerce REST API built with Spring Boot.

## Features
- Full CRUD for category management
- Pagination and sorting with query parameters
- Global exception handling with `@RestControllerAdvice`
- DTO pattern with ModelMapper
- Jakarta Bean Validation on request payloads

## Tech Stack
Spring Boot · JPA/Hibernate · PostgreSQL · Lombok · ModelMapper · Jakarta Validation

## Running Locally
```bash
./mvnw spring-boot:run
```
